Mathematical Logic

This blog is a record of my study of the book Mathematical Logic written by Bell.This is a relatively difficult textbook.And at the same time,this blog can be a outline of the knowledge of Mathematical Logic.

Basic concept

Before talking about Mathematical Logic, There are some basic concepts we have to mention about.

  • structure:
    Structures usually is symboled as \(\mu\) are mathematical objects or frames which consist of three components:

    1. universe(domain):A non-empty class.The members of the class are called individuals.

    2. operations

    3. relations

      examples:Elementary arithmetic,Set theory,and Topological space.

  • formal language:
    A formal language is symboled as \(\psi\),which is base on a structure.It is a set of strings which regularly consist of variables,function symbols,predicate symbols,etc.

  • metalanguage and object language:
    When discuss formal languages,we have to distinguish the Mathematical objects that we want research and the language we used to describe the object language and the later is called metalanguage.
    And in the second level,a symbol have two meanings:one is the object it denoting and one is the symbol itself.So,for distinguishing them,we use symbols wirtten by handwriting.

  • symbols(syntax) of the first order language

    1. variables: \(x,y,z \) or \(v_1 \).
    2. fuction symbols:
    3. predicate symbols:
    4. connectives: \(\ \) (negation) and (implication)
    5. The universal quantifier \( \)

  • terms,formulas,sentences:

    • A term is a string consists of a variable or a n-ary function symbol followed with n terms.(This kind of definition is by recusion)
    • A formula is a string consists of
      1. An n-ary predicate symbol followed with n terms.
      2. a implication followed with two formulas.
      3. a negation followed with a formulas
      4. A quantifier symbol followed with a variable and a formula.
    • A sentence is a formula whose variables are all bounded variable (will be defined later)

  • weight and degree of complexity:
    these conceptions can help us to decuce whether a strings is in a first order language or not precedurely.

  • prime formula:atom formula or universal formula.

First-order logic

Basic conceptions

  • syntax and semantics:
    A language is just a set of strings which fit with some rules.It is just something in formal,we say this is syntax.It becomes meaningful when we give the symbols meaning.And we call the given meaning in a string semantics.In the first-order logic,we use structure(interpretation) as the surppoter of the semantics.

    • each function symbol assign the meaning of a oparation in the structure
    • predicate symbols assign to relations
    • we define a valuation \(\sigma \):a map from variables to individuals.A valuation must underline a structure.A language with the same structure may have different valuations. note:valuation is different with truth valuation.

  • BSD(Basic Semantic Definition):

    • T1 T2
    • F1 - F4

Substitutiion

Substitution is changing one variable or terms,formulas as another expression.After this step,the semantics of the formulas or terms may be changed.But at some situations,under some conditions,the value do not change.This shows some variant properties in the language and also useful in simplifing strings.we write \( s(x/t)\) to express substituting \(t\) for each \(x\).

  • for \(\bf{s},\bf{t}\) are terms, \(\bf{x}\) a variable.
    \( \bf{s(x/t)^{\sigma}} = {\bf{s}}^{\bf{\sigma (x} / \it{t})} \), \( t=\bf{t} ^{\bf{\sigma}} \)

  • for \(\bf{\alpha}\) a formula, \(\bf{x}\) a variable,\(\bf{t}\) a term,we must

    1. define \( \bf{\alpha (x/t)}\) in a way that the substitution only change the free occurrence of \( \bf{x}\) in \(\bf{\alpha}\) to ensure the equation above satisfied.
    2. avoid free variables being bouned.
      so we define :
      \(\bf{t}\) is free to be substituted for \(\bf{x} \) in \( \bf{\alpha} \) if no free occurrence of \(\bf{x} \) in \( \bf{\alpha} \) is within a subformula of \( \bf{\alpha} \) having the form \(\bf{ \forall y \beta} \) , where \(\bf{y}\) occurs in \( \bf{t} \).
      And then we define substitution by induction such that the equation still satisfied.

  • And for dealing with the substitiution of not free like above,we define alphabetic change

  • finally,we can define the general substitution and prove that it preserve the value of the semantics.

  • And substitution can be extended to multi-situation.
